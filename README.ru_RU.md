# 3X-UI Fork: множитель трафика

Этот репозиторий является личным fork проекта 3X-UI. Он следует за upstream-проектом, а этот README описывает только основные функции, добавленные сверх upstream.

## Основная добавленная функция

### Множитель трафика для inbound

В этом fork добавлен множитель трафика для inbound-узлов.

- В формах Add Inbound и Edit Inbound добавлено поле Traffic Multiplier.
- Значение по умолчанию `1` сохраняет исходное поведение подсчета трафика.
- Если установить множитель `2`, входящий upload/download будет учитываться как двойной фактический расход.
- Upload/download отдельных клиентов внутри inbound также учитывается с этим множителем.
- Остаток трафика клиента и проверка превышения лимита используют взвешенные значения.
- Синхронизация inbound с удаленными узлами передает значение множителя.
- Subscription links не изменяются. Использованный и оставшийся трафик в subscription основаны на взвешенной статистике клиентов.
